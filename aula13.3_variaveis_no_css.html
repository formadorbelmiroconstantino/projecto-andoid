Variáveis no CSS

Variáveis CSS (ou propriedades personalizadas) permitem armazenar valores reutilizáveis (cores, fontes, etc.) em um único lugar, facilitando a manutenção e consistência do código, com sintaxe --nome-variavel: valor; para declarar e var(--nome-variavel) para usar, geralmente declaradas globalmente em :root ou localmente em selectores específicos, e são sensíveis a maiúsculas/minúsculas e acessíveis via DOM, possibilitando temas e actualizações dinâmicas com JavaScript.

Como declarar

  • Global (todo o site): Dentro do seletor :root.
    • :root {
    --cor-primaria: #A47864;
    --fonte-principal: Arial, sans-serif;
    }
  • Local (em um seletor específico):
    • .header {
    --cor-primaria: #ff5733; /* Apenas para o header */
    background-color: var(--cor-primaria);
    }

Como usar

  • Use a função var() e passe o nome da variável como argumento body {
    background-color: var(--cor-primaria); /*A declarada no :root{}*/
    font-family: var(--fonte-principal);
    }
